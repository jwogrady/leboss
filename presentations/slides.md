# LEBOSS Presentation — Canonical Archive

**Note:** The presentation source file lives at [`presentation/slides.md`](../presentation/slides.md).

The `presentation/` directory (singular) serves as both:
- The Netlify build root (configured in `netlify.toml`, published at https://leboss.status26.com/)
- The canonical preserved artifact of the original LEBOSS slide deck

This directory (`presentations/`, plural) exists as the archive reference path. The authoritative copy is in `presentation/slides.md`. Do not edit this file — edit the source there.

---

## About the Presentation

The LEBOSS slide deck is the historical origin of this standard. It was the first articulation of the LEBOSS concept — the six-element topology, the five Foundation Principles — before any formal specification existed.

It is preserved as a primary source. The standards documents in this repository are derived from it, not the other way around.

**To view the presentation locally:**

```bash
cd presentation
npm install
npm run dev
```

Then open http://localhost:3030
