# LEBOSS Architecture Deck

**Audience:** Developers, platform architects, technical implementers

**Purpose:** Explains the LEBOSS reference model (Universe through Satellite), governance objects (Access Grants, Integration Descriptors, Audit Records, Resources, Data Portability), and the operational flow.

## Slide source

The deck source is at [`../slidev/architecture.md`](../slidev/architecture.md).

It is located in the Slidev build environment directory so that theme and package resolution work correctly during build.

## Running locally

```bash
cd ../slidev
npm install
npm run build:architecture   # outputs to dist/architecture/
```

Or open `../slidev/architecture.md` with a local Slidev installation.
