# LEBOSS Governance Deck

**Audience:** Contributors, maintainers, committee members, implementers

**Purpose:** Explains the LEBOSS proposal lifecycle (Proposal → Draft → Committee Vote → Published), versioning model, committee roles, how to contribute, and conformance requirements.

## Slide source

The deck source is at [`../slidev/governance.md`](../slidev/governance.md).

It is located in the Slidev build environment directory so that theme and package resolution work correctly during build.

## Running locally

```bash
cd ../slidev
npm install
npm run build:governance    # outputs to dist/governance/
```

Or open `../slidev/governance.md` with a local Slidev installation.
