# LEBOSS Overview Deck

**Audience:** Local business owners, solution providers, developers evaluating LEBOSS

**Purpose:** Explains why LEBOSS exists, the data sovereignty problem, the five foundation principles, and the reference architecture.

## Slide source

The deck source is at [`../slidev/overview.md`](../slidev/overview.md).

It is located in the Slidev build environment directory so that theme and package resolution work correctly during build.

## Running locally

```bash
cd ../slidev
npm install
npm run dev        # serves this deck at http://localhost:3030
```

## Deployed

This deck is the primary Netlify deployment at **https://leboss.status26.com/**
